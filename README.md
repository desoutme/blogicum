# Blogicum

Современная платформа для публикации статей и общения пользователей, разработанная на **Django**. Проект демонстрирует навыки создания полнофункционального веб-приложения с использованием архитектурных возможностей Django, ORM, встроенной системы аутентификации и шаблонизатора.

---

## О проекте

Blogicum позволяет пользователям создавать публикации, делиться своими мыслями, комментировать записи других авторов и управлять собственным контентом. При разработке особое внимание уделялось разделению ответственности между компонентами приложения, удобству пользовательского интерфейса и соблюдению принципов Django.

---

## Основные возможности

- Регистрация и аутентификация пользователей
- Создание, редактирование и удаление публикаций
- Загрузка изображений к публикациям
- Комментирование записей
- Личный профиль пользователя
- Категории публикаций
- Привязка публикаций к локациям
- Постраничная навигация
- Разграничение прав доступа
- Подсчет количества комментариев
- Панель администратора Django

## Архитектура

Проект построен по классической архитектуре Django и состоит из нескольких взаимосвязанных приложений.

Основная бизнес-логика реализована в приложении **blog**, которое содержит модели, представления, формы и маршрутизацию.

### Основные сущности

- Пользователь
- Публикация
- Категория
- Локация
- Комментарий

Связи между моделями реализованы средствами Django ORM с использованием отношений **ForeignKey**.

---

## Реализованный функционал

### Работа с публикациями

- создание публикаций;
- редактирование собственных записей;
- удаление публикаций;
- прикрепление изображений;
- выбор категории;
- выбор локации;
- отображение только опубликованных записей;
- отображение публикаций в обратном хронологическом порядке.

### Комментарии

Пользователи могут:

- оставлять комментарии;
- редактировать собственные комментарии;
- удалять свои комментарии;
- просматривать количество комментариев под каждой публикацией.

### Пользовательские профили

Каждый пользователь имеет собственную страницу профиля, где отображаются:

- информация о пользователе;
- список опубликованных записей;
- количество публикаций.

## Запуск проекта

### Клонирование репозитория

```bash
git clone https://github.com/desoutme/blogicum.git
cd blogicum
```

### Создание виртуального окружения

Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

Windows

```powershell
python -m venv venv
venv\Scripts\activate
```

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Применение миграций

```bash
python manage.py migrate
```

### Создание администратора

```bash
python manage.py createsuperuser
```

### Запуск сервера

```bash
python manage.py runserver
```
